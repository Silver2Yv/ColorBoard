# 用户指南

## 界面概览

页面分为左右两部分：左侧为调色盘画布区域，右侧为控制面板。

控制面板从上到下包含：
- **模式切换按钮** — 2D / 3D
- **色彩空间切换按钮** — RGB / HSV / HSL
- **当前颜色色块预览**
- **颜色值显示** — HEX、RGB、HSV、HSL 四种格式

---

## 2D 模式操作

### RGB 模式

画布上显示一个彩色三角形，三个顶点分别为红、绿、蓝三原色，三角形内部通过重心坐标插值生成所有中间色。

- **点击** — 在三角形任意位置点击，选中该处颜色
- **拖动** — 在三角形内按住并拖动，颜色随鼠标位置实时变化
- 选中的位置会显示一个带黑色描边的白色圆形标记

### HSV 模式

画布分为左右两部分：
- **左侧矩形** — 横轴为色相（Hue, 0-360°），纵轴为饱和度（Saturation, 0-100%）
- **右侧滑块** — 控制明度（Value, 0-100%）

操作方式：
- **矩形区域** — 点击或拖动选择色相和饱和度
- **右侧滑块** — 点击或上下拖动调整明度
- 当前颜色的色相和饱和度位置用十字准线标记，明度位置用横线标记

### HSL 模式

布局与 HSV 模式相同，右侧滑块改为控制亮度（Lightness, 0-100%）。

- **左侧矩形** — 横轴为色相（Hue），纵轴为饱和度（Saturation）
- **右侧滑块** — 控制亮度（Lightness）

操作方式与 HSV 模式一致。

### 2D 模式通用说明

- 三个色彩空间共享同一个画布，切换空间时会自动重绘
- 所有操作实时生效，右侧面板的颜色值和色块同步更新

## 3D 模式操作

点击"3D"按钮切换到三维可视化模式。首次切换时会动态加载 Three.js 库，加载过程中画布保持空白，加载完成后自动显示当前色彩空间的 3D 图形。

### 通用操作

- **旋转** — 按住鼠标左键并拖动，从任意角度观察色彩空间
- **缩放** — 滚动滚轮拉近或拉远视角
- **点击选色** — 点击 3D 图形表面任意位置，选中该处颜色
- 选中位置会显示一个白色发光小球作为标记

### RGB 立方体

一个彩色立方体，三个轴分别对应 R、G、B 通道，坐标范围从 0 到 255。

- 原点 (0,0,0) 为黑色，对角点 (255,255,255) 为白色
- 三条边分别对应红、绿、蓝纯色渐变
- 立方体表面展示了 RGB 空间的全貌

### HSV 圆柱体

一个彩色圆柱体，直观展示 HSV 色彩空间结构。

- **圆周方向（色相 H）** — 环绕圆柱一周对应色相 0-360 度
- **半径方向（饱和度 S）** — 从中心到边缘，饱和度从 0 到 100%
- **高度方向（明度 V）** — 从底部到顶部，明度从 0 到 100%
- 中心轴线为灰度渐变

### HSL 双锥体

两个圆锥体底部相对拼接而成，展示 HSL 色彩空间结构。

- **圆周方向（色相 H）** — 环绕一周对应色相 0-360 度
- **半径方向（饱和度 S）** — 从中心到边缘，饱和度从 0 到 100%
- **高度方向（亮度 L）** — 从底部到顶部，亮度从 0 到 100%
- 顶部顶点为白色，底部顶点为黑色，中间赤道面为最饱和颜色

## 剖面模式

剖面模式是 3D 模式下的一项增强功能。通过锁定色彩空间中的一个坐标轴，生成该位置上的二维色彩切片，揭示三维色彩空间的内部结构。本功能特别适合需要精确选色或理解色彩空间维度关系的场景。

### 进入剖面模式

进入 3D 模式后，控制面板底部会自动显示"剖面模式"区域。该区域包含三个坐标轴（X/Y/Z）对应的行，每行由标签、滑块、数值输入和锁定按钮组成。剖面模式仅在 3D 模式下可用，切换到 2D 模式时自动隐藏。

### 锁定坐标轴

[截图：剖面模式控制面板，显示三个轴的锁定按钮和数值滑块]

点击某个轴的"锁定"按钮，即可锁定该轴的当前值：

- 锁定后，该轴的滑块和输入框变为不可操作状态，按钮文字变为"已锁定"
- 3D 场景中会出现一个垂直于该轴的半透明白色剖切平面，标明锁定位置
- 下方同步显示该位置上的二维色彩切片（2D Canvas 绘制）

不同色彩空间下坐标轴的含义：

| 色彩空间 | X 轴 | Y 轴 | Z 轴 |
|---------|------|------|------|
| RGB | R（红，0-255） | G（绿，0-255） | B（蓝，0-255） |
| HSV | H（色相，0-360°） | S（饱和度，0-100%） | V（明度，0-100%） |
| HSL | H（色相，0-360°） | S（饱和度，0-100%） | L（亮度，0-100%） |

[截图：锁定 R 轴后的 3D 场景，显示半透明剖切平面]

### 在剖面上选色

锁定轴后，剖面面板下方会出现对应位置的 2D 彩色切片：

- **点击** — 在切片任意位置点击，选中该处颜色
- **拖动** — 在切片上按住并拖动，颜色随鼠标位置实时变化
- 被锁定的轴的值保持不变，其他两个轴自由变化
- 选中位置显示白色圆形标记（黑色描边）

[截图：RGB 模式下锁定 R 轴后的 G-B 剖面切片]

各剖面切片的内容：

| 色彩空间 | 锁定轴 | 2D 切片内容 | 横向 | 纵向 |
|---------|--------|------------|------|------|
| RGB | R | G-B 平面 | B (0-255) | G (0-255) |
| RGB | G | R-B 平面 | R (0-255) | B (0-255) |
| RGB | B | R-G 平面 | R (0-255) | G (0-255) |
| HSV | H | S-V 平面 | S (0-100%) | V (0-100%) |
| HSV | S | H-V 平面 | H (0-360°) | V (0-100%) |
| HSV | V | H-S 平面 | H (0-360°) | S (0-100%) |
| HSL | H | S-L 平面 | S (0-100%) | L (0-100%) |
| HSL | S | H-L 平面 | H (0-360°) | L (0-100%) |
| HSL | L | H-S 平面 | H (0-360°) | S (0-100%) |

### 3D 剖切平面的意义

- **理解色彩空间结构** — 剖面揭示了固定一个维度时，剩余两个维度的颜色变化规律。例如在 HSV 圆柱体中锁定色相（H 轴），可以直观看到该色相下饱和度和明度的完整分布
- **精准选色** — 在三维空间中，剖面提供了类似 2D 平面式的选色体验，比在立体表面选色更精确，且支持在剖面上直接拖拽微调
- **教学辅助** — 帮助学生直观理解色彩空间维度之间的关系，例如在 RGB 立方体中锁定红色分量后观察绿色和蓝色的交互变化
- **设计参考** — 设计师可以在同一色相下快速浏览不同饱和度/明度的配色方案，或在同一亮度下比较不同色相

### 解锁与退出

- **解锁轴** — 再次点击已锁定的轴的"已锁定"按钮即可解锁，恢复 3D 场景正常操作，剖切平面消失
- **切换色彩空间** — 切换到其他色彩空间时，剖面面板的标签和取值范围自动更新，2D 切片和 3D 剖切平面同步切换
- **切换为 2D 模式** — 剖面面板自动隐藏

## 色彩空间切换

点击控制面板上的 RGB、HSV 或 HSL 按钮即可切换色彩空间。

- **2D 模式** — 切换后画布立即切换到对应的取色控件
- **3D 模式** — 切换后场景中的 3D 图形随之变化，视角保持不变
- 所有模式下，当前选中的颜色值在切换空间后仍然保持，不会丢失

### 色彩空间说明

| 空间 | 适用场景 | 说明 |
|------|---------|------|
| RGB | 屏幕显示、图形编程 | 基于红绿蓝三原色加色模型，每个通道 0-255 |
| HSV | 图形设计、颜色选取 | 色相(H)、饱和度(S)、明度(V)，更符合人类直觉 |
| HSL | 图形设计、数据可视化 | 色相(H)、饱和度(S)、亮度(L)，与 HSV 类似但亮度定义不同 |

## 触摸操作

本工具基于 Pointer Events 实现，兼容触摸屏设备。

### 2D 模式触摸操作

- **单指点击** — 选取颜色
- **单指拖动** — 在取色区域内滑动调整颜色
- 所有 2D 取色控件（三角形、矩形、滑块）均支持触摸操作

### 3D 模式触摸操作

- **单指拖动** — 旋转 3D 图形
- **双指捏合** — 缩放视角
- **单指点击** — 选取 3D 图形表面的颜色

### 响应式布局

- 屏幕宽度小于 768px 时，布局自动切换为上下排列
- 画布区域和面板区域各占一半高度

## 颜色值显示

右侧面板实时显示当前选中颜色的四种表示方式：

| 格式 | 示例 | 说明 |
|------|------|------|
| HEX | #FF5733 | 十六进制颜色码（大写，6 位） |
| RGB | 255, 87, 51 | 红、绿、蓝分量（各 0-255） |
| HSV | 10°, 80%, 100% | 色相(0-360°)、饱和度(0-100%)、明度(0-100%) |
| HSL | 10°, 100%, 60% | 色相(0-360°)、饱和度(0-100%)、亮度(0-100%) |

色块预览区域显示当前颜色的实际效果，方便直观比对。
